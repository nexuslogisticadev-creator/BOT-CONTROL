Este arquivo contém uma descrição resumida do projeto BOT-CONTROL, parte do portfólio de Adiel Alves.

## BOT-CONTROL

Sistema de automação e painel para gestão de entregas, motoboys e controle financeiro.

- Interface gráfica otimizada
- Automação de leitura/escrita de Excel
- Monitoramento de pedidos e pagamentos
- Gestão de vales e descontos
- Notificações e integração com Telegram

Para detalhes completos, acesse o README do projeto na pasta `BOT-CONTROL` ou [diretamente aqui](https://github.com/nexuslogisticadev-creator/BOT-CONTROL).
